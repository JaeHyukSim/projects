## 개발 일정

  

  

## 주제 선정 계기

  

***구매 결정 연기***  : 장바구니에만 잔뜩 담아놓고 안 사는 사람들의 심리

-> 쇼핑몰은 이러한 구매 결정 연기를 줄이고 싶어 한다.

  

이 부분을 ***'소비자 선택 제한'*** 마케팅 기법으로 해소해보자!

  

기능 1) 연관분석을 통해 사용자게에 맞는 상품을 추천한다.

기능 2) 상품을 보면서 사용자가 느끼는 감성을 실시간으로 분석해 시각화하여 제공한다.
